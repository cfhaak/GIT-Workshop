# GIT-Workshop  
Materialien für eine Einführung in Git mit geringen Vorkenntnissen.

Ich stelle die Funktionen von Git in einem graphical user interface vor. Diese Intefaces sind sehr nützlich, wir müssen nicht mit dem Terminal durch Ordner navigieren und uns keine komplizierten Befehle merken. Einige – mich eingeschlossen –bevorzugen allerdings CLIs (command line interfaces) die in einem Terminal ausgeführt werden. Die untenstehende Liste enhält neben allgemeinen Quellen auch spezifisches Material über die Nutzung von CLIs.

* [Grundlagen von CLI und Terminal befinden sich hier (EN)](shell_crash_course.md)  
* [Einführende, vertiefende und weiterführende Informationen über Git finden sich in diesem sehr guten Buch (CLI-basiert).](https://git-scm.com/book/de/v2)  
* [Für fortgeschrittene Techniken rund um Branches empfiehlt sich diese Ressource (setzt Kenntnisse des CLI voraus).](https://learngitbranching.js.org/?locale=de_DE)
